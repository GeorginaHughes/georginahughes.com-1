Profile

I graduated from Oxford Brookes University with a BSc Honours in Computer Science. I’m confident using Windows and Mac OS X, and have some experience of Ubuntu.

Since leaving university I have been professionally employed as a web developer, specialising in HTML, CSS and JavaScript (all, 7 years). I have also been using HTML 5, CSS 3 (both 4 years), AngularJs, Grunt, Bower, and Node (remainder, 2 years) in my recent projects.

I’m confident using server side technologies such as C# .NET (4 years), IIS (4 years), and PowerShell (1 year). I have worked on sites that were standalone, as well as those that were powered by Umbraco (4 years).

I have used both TFS (4 years) and Git (1 year) source control systems. I have been working in Agile environments for four years, with experience using Kanban and Scrum methodologies.

I have varying experience using T-SQL, MySQL, and SQLite. My highest profile projects to date are the 2012 adaptive website for ITN, and the 2014 re-launch of betway.com.
